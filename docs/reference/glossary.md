# Glossary

| Term | Meaning |
|------|---------|
| Milestone | A shippable product increment composed of multiple MVPs |
| MVP | A standalone vertical slice with a `SPEC.md` and independent app workspace |
| SPEC | The MVP proposal and implementation contract |
| Standalone MVP app | Isolated app under `apps/mvps/` used to build/review one MVP |
| Milestone integration app | App under `apps/milestones/` where completed MVPs are merged before production |
| Production app | The main app deployed to Vercel |
